# Bliss Frontend

## Overview
Bliss Frontend is a Flutter application designed to provide a seamless and professional user experience. The application utilizes a modern UI theme and is structured to facilitate easy navigation and component reuse.

## Project Structure
The project is organized into the following directories and files:

```
bliss-frontend
├── android                # Android platform-specific code
├── ios                    # iOS platform-specific code
├── lib                    # Main application code
│   ├── main.dart          # Entry point of the Flutter application
│   ├── theme.dart         # Application theme definitions
│   ├── screens            # Contains various screen widgets
│   ├── widgets            # Contains reusable widget components
│   └── models             # Contains data models
├── pubspec.yaml           # Project configuration and dependencies
└── README.md              # Project documentation
```

## Getting Started

### Prerequisites
- Flutter SDK installed on your machine.
- An IDE such as Android Studio, Visual Studio Code, or IntelliJ IDEA.

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/bliss-recruitment/bliss-frontend.git
   ```
2. Navigate to the project directory:
   ```
   cd bliss-frontend
   ```
3. Install the dependencies:
   ```
   flutter pub get
   ```

### Running the Application
To run the application, use the following command:
```
flutter run
```

## Features
- Light and dark theme support.
- Modular architecture with reusable components.
- Responsive design for various screen sizes.
